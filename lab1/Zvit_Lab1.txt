Контрольні запитання до лабораторної роботи #1 
	1. PyCharm - це программа для написання та тестування коду на Python
	2. СКВ - це інструмент для відстеження змін у коді та інших файлах проекту.
	   Різниця між SVN і Git:  
	   	SVN — це централізована система, де всі зміни зберігаються на одному сервері, де потрібен доступ до цього сервера.
		Git — це розподілена система, де кожен має повну копію проєкту, і можна працювати офлайн, а зміни синхронізуються пізніше.
	3.Git починає відстежувати зміни у зазначених файлах. Це означає, що зміни, які ви додали, будуть збережені під час наступного коміту.
	4.Команда git commit зберігає зміни, які ви додали в індекс, у вигляді нової версії
	5.Команда git push origin main (або git push origin <branch_name>) відправляє ваші зміни на віддалений репозиторій на GitHub, завантажуючи туди вашу останню версію проєкту.
